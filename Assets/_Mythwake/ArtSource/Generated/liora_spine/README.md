# Liora Spine Character Source

Generated original Mythwake chibi lightblade character. Runtime files live in `Assets/_Mythwake/Resources/Mythwake/Art/Spine/Liora/` and include:

- `hero_liora_spine.json` with `idle`, `run`, `attack`, `skill`, and `death` animations.
- `hero_liora_spine.atlas.txt` and `hero_liora_spine_atlas.png` for spine-unity runtime loading.
- `hero_liora.png` runtime portrait fallback under `Resources/Mythwake/Art/Runtime/`.

The art is original procedural chibi fantasy work, not copied from Ravik assets.
