# Mythwake Runtime Art Sources

This document tracks the temporary free art used for the playable Unity test stand. These are placeholder production assets, not final Mythwake branding.

## Current Runtime Asset Set

The curated PNGs under `Assets/_Mythwake/Resources/Mythwake/Art/Runtime/` are sourced from Kenney packs released as Creative Commons Zero (CC0). They can be used in personal, educational, and commercial projects. Credit to Kenney is appreciated but not mandatory.

- `Tiny Dungeon` by Kenney: placeholder heroes, enemies, dungeon markers, and item/currency icons.
- `Background Elements` by Kenney: layered fantasy backdrop pieces.
- `Particle Pack` by Kenney: slash, magic, and summon VFX textures.
- `UI Pack` and `UI Pack (RPG Expansion)` by Kenney: button and panel styling textures.

## Repo Policy

Do not commit raw downloaded zips or full extracted third-party packs. Keep those in a local cache only. Commit only curated, renamed runtime assets that are actually referenced by code or scenes.
